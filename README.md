# 📡 LoveLink Backend

This is the backend for **LoveLink**, a matrimonial platform that connects individuals looking for a life partner. The backend is built using **Node.js, Express.js, and MongoDB**.

---

## 🚀 Live API  
🔗 **Base URL:** `https://love-link-backend.vercel.app`  

---

## 📌 Features  
✅ RESTful API for biodata management  
✅ User authentication with JWT  
✅ Secure admin panel access  
✅ Contact purchase & payment integration  
✅ Biodata filtering & pagination  
✅ Admin dashboard insights  

---

## 🛠️ Tech Stack  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT  
- **API Calls:** Axios  


💙 **Thank you for using LoveLink!**
